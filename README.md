Este projeto é uma página simples de contagem regressiva para anunciar uma live ou evento que acontecerá em breve.
A interface apresenta:

Um título destacando a live;

Um contador mostrando dias, horas, minutos e segundos restantes até a data programada;

Um botão de cadastro para usuários interessados;

Layout estilizado com CSS e tipografia Montserrat via Google Fonts.

A contagem regressiva é controlada pelo arquivo main.js, que atualiza o tempo dinamicamente na página.
